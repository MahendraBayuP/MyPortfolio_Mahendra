IoT-Based Home Security System

The Home Security System utilizes MQTT implementation. This system aims to enable homeowners to monitor and control their home security effectively. We designed a prototype as a visualization of how the system will function using Wokwi. The core components of this system include the ESP32, an Ultrasonic Distance Sensor, and a Photoresistor/LDR Sensor for light detection. For security indications, we use LED indicator lights and a sound alert represented by a Buzzer Alarm. MQTT serves as the broker, facilitating message transfer from publishers to subscribers.
